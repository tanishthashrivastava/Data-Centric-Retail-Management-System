# Data-Centric-Retail-Management-System

##  Overview
A SQL-based retail management system designed to manage customers, products, orders, payments, and shipping operations with a strong focus on data analytics and automation.

## Technologies Used
- SQL Server
- Relational Database Design
- Stored Procedures
- Triggers
- Views
- Analytical SQL Queries

## Key Features
- Normalized database schema for retail operations
- Automated inventory updates using triggers
- Stored procedures for order processing
- Analytical queries for sales trends, customer insights, and KPIs
- Data integrity using constraints and cascading rules

## Business Use-Cases
- Sales trend and revenue analysis
- Inventory optimization
- Customer lifetime value analysis
- Payment and shipping performance tracking

## How to Run
1. Open SQL Server Management Studio
2. Execute `data_centric_retail_management_system.sql`
3. All tables, triggers, and queries will be created
